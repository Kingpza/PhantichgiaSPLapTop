# Đề tài: Phân tích giá sản phẩm Laptop tại Phong Vũ

# Cấu trúc dự án
- `thuthapdulieu.ipynb`: Mã nguồn sử dụng Selenium để quét và tải dữ liệu tên, giá, link sản phẩm.
- `Laptop_PhongVu.csv`: Tập dữ liệu thô (Dataset) chứa danh sách các laptop thu thập được.
- `phantichgiasp.ipynb`: Mã nguồn sử dụng Pandas, Matplotlib và Seaborn để xử lý dữ liệu và vẽ biểu đồ.
- `requirements.txt`: Danh sách các thư viện Python cần thiết để chạy dự án.
